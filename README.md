<h2>📖 프로젝트 개요</h2> 
<p>[Problem] 많은 사람들이 집에서 혹은 혼자 공부할 때 동기부여가 부족하여 공부를 지속하기 힘들다는 점에 주목</p>
<p>[Solution] 온라인에서 접속하여 함께 공부할 수 있는 스터디룸을 만들어보자는 아이디어를 구상하게 됨</p>

<br/><br/>
<h2> 🚀 MVP 핵심 기능</h2>

1. **소셜 로그인 및 보안**
   - SSL 인증으로 HTTPS 적용
   - Google, Kakao, Naver OAuth2.0 로그인 및 JWT 기반 인증 시스템 
   
2. **실시간 스터디룸**
   - Web RTC를 활용한 화상통화, 화면공유, 실시간 채팅 기능과 방 생성/입장 시스템 및 비밀번호를 통한 제한 입장 기능

3. **질문 게시판**
   - 게시글 작성/수정/삭제, 댓글 및 대댓글 시스템, 좋아요 기능과 페이지네이션 적용
     
4. **공부 시간 관리, D-Day, 목표 설정 기능**
   - 실시간 타이머, 목표 시간 설정, 오늘/누적 공부 시간 통계 및 진행률 표시, 시험일정 D-Day 계산, 개인 각오 설정, 목표 달성률 시각화
     
5. **다크/라이트 테마 및 반응형 UI/UX**
   - 사용자 편의를 위한 테마 전환 기능 및 반응형 디자인을 통한 모바일 환경 제공
